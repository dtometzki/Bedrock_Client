# Bedrock Client

Interactive CLI client for AWS Bedrock.

## Requirements

- Node.js 20+
- AWS CLI installed and configured

## Install

```bash
npm install
```

## Run

```bash
node app_aws.js
```

Or link it as a CLI:

```bash
npm link
bedrock-chat
```

## Build

```bash
npm run build
./dist/bedrock-chat
```
